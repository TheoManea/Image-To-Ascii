# Image To Ascii converter writter in Python
 
 ## How it works ?
  
 The original picture is analysed by the PIL library which returns a pixel matrix.
 Then each pixel is remplaces by one of those special character : `["@", "J", "D", "%", "*", "P", "+", "Y", "$", ",", "."]`
